# Brand Identity

## TL;DR
Qvora = "The force that devours mediocre ads." Bold, sharp, performance-obsessed brand. Primary CTA color is Volt purple (#7B2FFF). Hero tagline: "Born to Convert."

## Design Tokens

```css
--color-volt: #7B2FFF;           /* Primary brand, CTAs */
--color-convert-green: #00E87A;  /* Success, performance wins */
--color-signal-red: #FF3D3D;     /* Fatigue alerts, warnings */
--color-data-blue: #2E9CFF;      /* Analytics, charts */
--color-qvora-black: #0A0A0F;    /* Primary background */
--color-qvora-white: #F5F5F7;    /* Text, surfaces */
```

**Usage rules:**
- Volt = CTAs, primary interactive elements, brand moments
- Convert Green = positive metrics, success states, wins
- Signal Red = fatigue warnings, errors, urgent alerts
- Data Blue = charts, analytics, data-heavy views
- Black = page background (dark-first brand)
- White = text and surface color on dark backgrounds

## Typography
