# Personal Notes

This section is for founder notes, goals, reflections, lessons learned, and anything else that doesn't fit cleanly into the product, market, or architecture domains.

## Suggested Files to Create

- `goals.md` — current goals, OKRs, what you're optimizing for
- `weekly-notes.md` — weekly reflections, wins, blockers, course corrections
- `lessons.md` — things you learned the hard way (product, technical, market)
- `hypotheses.md` — product/market bets you're testing; mark them as confirmed/refuted as evidence comes in

## How to Use

Paste journal entries, voice memos, or rough notes and ask the LLM to:
1. Extract key insights and file them to the right page
2. Update `hypotheses.md` if new evidence confirms or refutes a bet
3. Surface connections to market or product pages (e.g., "your note about agency creative cycles connects to [[market-context]]")
