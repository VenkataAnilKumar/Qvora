## 1. Market Context

### Why This Market, Why Now

Performance marketers are operating in an increasingly hostile cost environment:

| Platform | Average CPM (2026) | Trend |
|---|---|---|
| Instagram | $9.46 | ↑ Rising |
| TikTok | $4 – $9 | ↑ Rising |
| YouTube | $4 – $5 | ↑ Rising |
| Pinterest | $4.67 | ↑ Rising |

*Source: Hootsuite Social Media Advertising Report, 2026*

CPMs are rising across every major platform — meaning **the same ad budget reaches fewer people every quarter**. The only lever performance teams can pull without increasing spend is **creative quality and velocity**: producing more testable, platform-native variants faster than the competition.

### The Creative Velocity Problem

The fastest-growing unit in performance marketing is **creative velocity** — the ability to produce, test, and learn from ad variants faster than competitors.

Current state for most performance teams:
- 3–5 day cycle from brief to approved ad set
- 20–40 creative variants tested per month; teams need 80+
- Performance data lives in one tool, creative in another — no connection
- Teams keep running dead creatives because fatigue detection is manual
- No tool closes the **brief → production → performance feedback → next iteration** loop autonomously

**Platform case study:** PureGym ran lo-fi, native-style Reels ads designed to blend into the feed. This approach drove a **5.6x increase in Thruplays** over their previous polished ads — and was faster and cheaper to produce. *(Source: Hootsuite / Meta, 2026)*

## 2.5 AI Video Generation Model Landscape (Infrastructure Layer)

> These are not direct competitors to Qvora — they are the **model infrastructure layer** that Qvora routes through via FAL.AI unified API. Understanding this landscape informs Qvora's model strategy and cost structure.

### Key Market Event — Sora 1 Shutdown (March 24, 2026)
OpenAI shut down Sora 1 on March 24, 2026. Downloads had declined 45% and Disney cancelled a planned $1B investment. OpenAI redirected compute to their next LLM. Sora 2 remains available. **Implication for Qvora:** Any positioning built around Sora 1 is obsolete. Qvora's model-agnostic routing via FAL.AI insulates against individual model shutdowns — this is a design advantage, not a risk.

### Model Comparison

| Model | Provider | Strength | Duration | Cost Est. | Qvora Use |
|---|---|---|---|---|---|
| **Veo 3.1** | Google | 4K 60fps, native audio (SFX + ambient), tops T2V leaderboards | Up to 60s | $0.05–$0.40/sec | Premium T2V, brand cinematic |
| **Kling 3.0** | Kuaishou | Longest clips (up to 2 min), best cost/quality ratio, best I2V | Up to 2 min | $6.99/mo or API | Default T2V + I2V |
| **Runway Gen-4.5** | Runway | Best camera control (pan/zoom/dolly/static), professional standard | Up to 60s | Usage-based | T2V with camera direction |
| **Sora 2** | OpenAI | Cinematic physics, synchronized audio, high prompt adherence | ~30s | Usage-based | High-fidelity T2V |
| **Stable Video Diffusion** | Stability AI | Open-source, customizable, ControlNet support | Short clips | Self-hosted option | I2V fallback / cost control |
| **HeyGen Avatar v3** | HeyGen | #1 photorealistic lip-sync, full-body motion, 175+ languages | Up to 5 min | API credits | All V2V generation |

### API Routing Strategy
- **Primary API layer:** FAL.AI unified API — access to 600+ models including all above at $0.05–$0.40/sec
- **V2V dedicated:** HeyGen Avatar v3 API (separate integration; premium lip-sync quality)
- **Fallback logic:** If primary model unavailable (outage/rate limit), auto-route to next preferred model per use case
- **Cost ceiling:** Average generation COGS ≤ $0.40/video to maintain unit economics at Starter tier ($99/mo, 20 ads)

## 6. Go-to-Market Opportunity

### The White Space
No product currently closes the full loop:

```
Brief → Creative Strategy → Production → Structured Testing → Performance Data → Better Brief
```

Creatify covers Production → Structured Testing (partially) → Performance Data.  
Arcads covers Production (actor-first).  
HeyGen covers Production (avatar-first).  

**Qvora covers the entire chain** — and is the only tool that starts with intelligence before a single pixel is rendered.

### Wedge Strategy
- **Phase 1 (Launch):** Performance marketing agencies — multi-brand accounts, high creative volume, willing to pay $249–$499/mo, network effects to peer agencies
- **Phase 2:** DTC brands ($5K–$50K/mo ad spend) — follow agency adoption, budget $100–$500/mo
- **Phase 3:** Mobile App UA Managers — requires MMP integration (AppsFlyer/Adjust); unlock after V2 Signal is live

### Competitive Moat Timeline
| Horizon | Moat |
|---|---|
| **V1 (Launch)** | Strategy layer — brief, angles, hooks before any generation |
| **V2 (6 months)** | Performance Learning Engine — ad performance re-trains the brief |
| **V3+ (12 months)** | Proprietary creative intelligence data across all agency accounts |
